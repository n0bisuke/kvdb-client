# examples

Set env vars:

```bash
export KVDB_BUCKET=YOUR_BUCKET
export KVDB_TOKEN=YOUR_TOKEN # optional for public buckets
export KVDB_EMAIL=you@example.com
```

Run:

```bash
node examples/create-bucket.js
node examples/basic.js
node examples/counter.js
node examples/transaction.js
node examples/auth.js
```
